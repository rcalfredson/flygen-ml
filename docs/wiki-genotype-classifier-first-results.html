High-level summary

  • Scientific question: do movement patterns in between-reward trajectories contain enough information to distinguish fly genotype, specifically Control>Kir versus PFN>Kir?
  • Short answer: yes, as a first proof of concept. A movement-only fly-level logistic regression classifier reached 75.4% validation accuracy overall, and 87.8% accuracy among high-evidence flies with at least 50 between-reward segments.
  • Interpretation: between-reward trajectory summaries appear to contain genotype-relevant signal. A grouped cross-validation check supports this conclusion, while also showing meaningful fold-to-fold variability.

 

Dataset and pipeline

This first classifier uses the current fly-level engineered feature table from Training 2 between-reward trajectory segments.

  • Code repository: flygen-ml on GitHub
  • Input feature table: artifacts/features_no_training_end.csv
  • Reference run: runs/logreg_v1_movement_only
  • Model config: configs/model/logreg.yaml
  • Unit of prediction: one experimental fly
  • Labels: Control>Kir versus PFN>Kir
  • Split: grouped by fly ID, so the same fly cannot appear in both train and validation

 

The pipeline currently runs end to end:

  1. glob-based manifest construction
  2. between-reward segment extraction
  3. fly-level engineered feature building
  4. logistic-regression baseline training
  5. misclassification and segment-level inspection

Analysis choices

Why terminal training-end segments were excluded

Segment extraction keeps all segments, including segments that are terminated only because the selected training window ends. For feature building, these terminal segments are omitted by default because they do not end with reward re-entry. Including them would mix canonical reward-to-next-reward excursions with censored trajectory fragments whose endpoint has a different meaning.

 

Why segment count was excluded from model inputs

The movement-only model excludes n_segments and n_segments_with_qc_flags from the regression inputs. This keeps segment count from becoming a direct classifier feature. Instead, segment count is used as an evidence-strength descriptor for interpreting how reliable each fly-level feature summary is likely to be.

Main result: movement-only logistic regression

  • Validation n: 57 flies
  • Accuracy: 0.7544
  • Balanced accuracy: 0.7311
  • Control>Kir recall: 0.8788
  • PFN>Kir recall: 0.5833

 

This is a credible first proof of concept that movement-pattern features from between-reward trajectories contain genotype-relevant information. The asymmetry in recall suggests that the current baseline more reliably identifies Control>Kir than PFN>Kir, so the result is promising but not yet balanced across classes.

Evidence-bin result

Because fly-level features summarize a variable number of between-reward segments, validation performance was also reported by evidence strength.

  • High evidence, n_segments >= 50:
    • n = 41
    • accuracy = 0.8780
    • balanced accuracy = 0.8386
    • Control>Kir recall = 0.9630
    • PFN>Kir recall = 0.7143
  • Moderate evidence, 20 <= n_segments < 50: n = 10, accuracy = 0.5000, balanced accuracy = 0.5000
  • Low evidence, n_segments < 20: n = 6, accuracy = 0.3333, balanced accuracy = 0.2000

 

The strongest result is in high-evidence flies. This makes intuitive sense: with at least 50 reward-to-reward segments, the fly-level movement summaries are based on substantially more behavioral evidence. Low-segment-count flies should be treated cautiously because their feature summaries are sparse and likely noisier.

Cross-validation check

As a first stability check, the same movement-only logistic regression was evaluated with 5-fold grouped, label-aware cross-validation. Each fold held out flies not seen during training.

  • Overall validation accuracy: mean = 0.7380, SD = 0.0440, range = 0.6667-0.8000
  • Overall balanced accuracy: mean = 0.7237, SD = 0.0553, range = 0.6407-0.8057
  • High-evidence accuracy, n_segments >= 50: mean = 0.8038, SD = 0.0798, range = 0.6757-0.9167
  • High-evidence balanced accuracy, n_segments >= 50: mean = 0.7576, SD = 0.0857, range = 0.6473-0.9062

 

The cross-validation results support the main proof-of-concept conclusion: movement-only between-reward features contain genotype-relevant signal. However, they also temper the single-split result. High-evidence flies usually perform better than the full validation set, but performance varies across folds, and PFN>Kir recall is the least stable component.

Caveats

  • Fold-to-fold variability: the grouped cross-validation check shows that performance is not unique to one split, but also that the high-evidence result is not uniformly strong across folds.
  • Small evidence-bin counts: moderate- and low-evidence bins have only 10 and 6 validation flies, respectively, so their estimates are especially unstable.
  • Fly-level aggregation may hide structure: averaging segment features per fly is useful for a first baseline, but it discards the distribution and sequence structure of individual segments.
  • Class recall is uneven: PFN>Kir recall is lower and less stable than Control>Kir recall, including in the high-evidence subset.

Working conclusion

The current result supports the idea that between-reward movement trajectories carry genotype-relevant information. The grouped cross-validation check confirms that the signal is not unique to one train/validation split, but it also shows meaningful fold-to-fold variability. The most defensible near-term statement is: movement-only fly-level features provide a promising first genotype classifier, especially for high-evidence flies, but the model is not yet stable or balanced enough to treat as a final classifier.

Next steps

  1. Inspect fold-to-fold variability: identify which flies or recording conditions drive weak folds, especially folds with low PFN>Kir recall.
  2. Expand to the full four-group problem: train models that distinguish the four combinations of antenna condition and genotype: intact Control>Kir, intact PFN>Kir, antennae-removed Control>Kir, and antennae-removed PFN>Kir. This would test whether between-reward movement features capture genotype effects, antenna-removal effects, and their possible interaction.
  3. Move from per-fly to per-segment modeling: train models that see individual segments, then aggregate evidence to the fly level. This should preserve more behavioral structure than simple fly-level means.
  4. Try stronger model classes: compare logistic regression against modern methods such as tree ensembles, regularized linear models with richer features, and sequence or pooling models over segment-level trajectories.
